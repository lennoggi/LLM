# LLM
A large language model (LLM) built from scratch

## Instructions
- Build with
  ```
  ./build_all.sh
  ```
- Run with
  ```
  ./install/bin/llm
  ```
- Remove the build and install directories with
  ```
  ./clean_all.sh
  ```

## References
Raschka, Sebastian. *Build a Large Language Model (From Scratch)*. Manning Publications, 2024
